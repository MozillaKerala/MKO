- AngularJS (Frontend Framework)
- SailsJS (Backend Framework)

  - EJS (Template Engine)

- MariaDB (Database)
- Redis (Web storage)
- Forever (Production server deamon)
- Bootstrap (CSS Framework)
- Coffee Script (Optional suggestion)
